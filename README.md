# 🎮 DEV102 MemoryGame

Welcome to DEV102 MemoryGame - I hope you enjoy it 🚀

---

## 🧠 About the Project

DEV102 MemoryGame is a fun and interactive memory game developed to improve concentration and memory skills.

Flip the cards, find matching pairs, and try to finish with the least number of attempts!

---

## ✨ Features

- 🃏 Card matching system  
- 🧮 Score tracking  
- 🏆 Best score saved  
- 🎨 Simple and clean interface  
- ⚡ Fast and responsive gameplay  

---

## 🖥️ Technologies Used

- Python 🐍  
- Tkinter (GUI)  
- Pillow (Image processing)  
- Random module  

---

## 🎮 How to Play

1. Click on a card to reveal it  
2. Click on another card to find its pair  
3. Match all cards  
4. Try to win with minimum attempts
